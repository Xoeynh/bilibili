## 环境要求

需要 NodeJS 16.8.0+ 环境

![](./public/design-sketch/npm-7.21.0.svg)
![](./public/design-sketch/node-16.8.0.svg)

## 简介

本项目是一个仿写哔哩哔哩H5移动端项目

接口及接口文档请查看[bilibili-api](https://github.com/xlz122/bilibili-api)

## 项目使用技术栈

* React全家桶
* Next
* Redux
* TypeScript

## 加入项目

如果想加入本项目开发，需将项目fork到自己的仓库下，功能开发完成后，提PR即可

## 运行

启动项目

```
yarn install
```

```
yarn dev
```

项目打包

```
yarn build
```
