## 简介

本项目是一个仿写B站web移动端项目

## 项目使用技术栈

* React全家桶
* Next
* Redux
* TypeScript

## 加入项目

如果想加入本项目开发，需将项目fork到自己的仓库下，功能开发完成后，提PR即可

## 运行

启动项目

```
yarn install
```

```
yarn dev
```

项目打包

```
yarn build
```
